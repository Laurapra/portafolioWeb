Sistema de Monitoreo Climático para Cundinamarca

Desarrollé un sistema de Monitoreo Ambiental enfocado en el análisis de patrones climáticos en la región de Cundinamarca y Boyacá. Este proyecto busca identificar cambios y anomalías climáticas que podrían impactar la biodiversidad local.


    Descripción del Proyecto

    El sistema utiliza registros climáticos diarios (temperatura, precipitación, humedad) obtenidos de fuentes abiertas y datos simulados para el año 2013, específicamente para ciudades como Girardot, Soacha, Facatativá y Chocontá. Mi proyecto incluye:

  • Preprocesamiento de datos: Conversión de tipos de datos, manejo de valores faltantes mediante generación de datos sintéticos y filtrado por ubicación para asegurar la calidad y la consistencia del dataset.
  • Análisis Exploratorio de Datos (EDA): Creación de gráficos de líneas por tendencias diarias de temperatura, boxplots para variabilidad estacional y mapas de calor para identificar correlaciones entre variables climáticas.
  • Detección de Anomalías: Implementación de un método basado en promedio yu desviación estándar para identificar días con temperaturas inusualmente altas o bajas, señalando posibles eventos climáticos externos.

      • Habilidades Clave
    • Análisis de Datos: Preprocesamiento, limpieza y preparación de datasets complejos.
    • Visualización de Datos: Creación de gráficos informativos para comunicar insights climáticos.
    • Estadística: Aplicación de métodos para la detección de anomalías y análisis de tendencias.
    • Manejo de Datos Climáticos: Experiencia con series temporales y variables ambientales.

    Este proyecto demuestra mi capacidad para transformar datos climáticos en información accionable, crucial para la mitigación de impactos ambientales en ecosistemas vulnerables.
